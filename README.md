## Open ComfyUI Web UI
```
http://localhost:8188
``` 

## Run ComfyUI
```
sudo systemctl start comfyui
```

## Stop ComfyUI
```
sudo systemctl stop comfyui
```

## Restart ComfyUI
```
sudo systemctl restart comfyui
```

## Frontend

**Start frontend:**
```
cd frontend
npm run dev
```

**Launch frontend:**
```
http://localhost:5174/
```
